## Запуск

1. Запустить DBeaver 
2. Открыть SQL скрипт 
   ```sh
   Панель инструментов -> SQL -> Открыть SQL скрипт
   ```
3. Открыть файл Laboratory.txt.sql 
4. Скопировать содержимое файла Laboratory.txt.sql в скрипт 
5. Выполнить SQL запрос
